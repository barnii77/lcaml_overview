Getting Started

To start coding in LCaml, you'll need:

  1. An LCaml interpreter (download from github repo)
  2. A text editor or IDE
  3. Reference materials and tutorials (None exist as of 31.01.2024)

Sample Code


    let factorial = |n| {
      if (n <= 0) {
        return 1;
      } else {
        return n * factorial (n - 1);
      }
    };

    let value = 5;
    let fact = factorial value;
    return fact;  -- return 120 as result of file
